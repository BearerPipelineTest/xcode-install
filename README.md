# Xcode::Install

⚠️WIP⚠️

Install and update your Xcodes.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'xcode-install'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install xcode-install
